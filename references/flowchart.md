数量规则一览

  brief │ 固定 3 条 │ front_page + ai_keyword │ min_points=30   │ isNew=false（只读不占）
  trend │  1-3 条  │ front_page + ai_keyword + dev_keyword │ 200 │ isNew=false used=true
  tool  │ 固定 1 条 │ show_hn + dev_keyword（禁 front_page）│ min_points=2 │ isNew=false used=true

  出库顺序：brief → trend → tool（brief 先挑，不消耗）

输出文件命名规则

  卡片截图：{序号}_{分类}_{页码}_{标签}.png
  示例：01_商业趋势_P1_封面.png
        02_工具推荐_P2_详情.png
        03_每日简讯_P3_金句.png

  公众号文章：{系列}_article.html
  示例：brief_article.html
        trend_article.html
        tool_article.html

  参考素材：{系列}_{编号}.md
  示例：brief_01.md
        trend_01.md
        tool_01.md
